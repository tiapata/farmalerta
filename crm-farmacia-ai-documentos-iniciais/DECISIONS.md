# Architecture Decision Records

## ADR-001
A plataforma NÃO é um ERP.

## ADR-002
Integração somente leitura.

## ADR-003
Middleware Universal independente do CRM.

## ADR-004
Modelo Canônico obrigatório para todos os ERPs.

## ADR-005
Prioridade de integração:
1 API
2 Exportação
3 Banco
4 XML NFC-e
5 RPA

## ADR-006
WhatsApp é o canal oficial de comunicação.

## ADR-007
Push Notification apenas para campanhas rápidas, cashback e lembretes.

## ADR-008
Toda campanha pode utilizar IA para gerar banner, textos e CTA.

## ADR-009
Arquitetura SaaS Multi-Tenant.

## ADR-010
CRM sempre trabalha sobre dados sincronizados, nunca diretamente sobre o ERP.
